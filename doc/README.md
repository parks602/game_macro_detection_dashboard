# 대시보드 주요 내용

## 로그인 기능을 통한 권한 관리
- 계정 정보 DB화
- 비밀번호 저장 규칙 (보안 Unit 추천: SHA-2 기법)

## 페이지 별 의의

### RO1 매크로 현황
- 분석에 사용된 유저 및 결과 확인 가능
- 블럭 유저(billing DB 데이터 수령) 확인을 통한 매크로 유저 관리 실태 확인

### 유저별 매크로 탐지 근거
- 4개의 주요 기법 중 3가지 기법에서 탐지될 경우 매크로 유저로 분류
- 매크로 유저로 탐지된 유저의 근거 제시

#### (개발 필요) 매크로 검출 기법 적용에 따른 근거 데이터 추출 페이지
- 보안 Unit > 기안 없이 직접적인 다운로드는 불가

## 대시보드 요약

### 분석 기법에 대한 전체적인 설명
- 매크로 현황에 더해 서버 내 매크로 유저 관리 실태 진단

### 각 기법에 적용 결과 페이지
- **멀티 클라이언트 (같은 행위)**
- **멀티 클라이언트 (다른 행위)**
- **유저 간 유사도 비교**
- **유저별 자기 유사도 비교**
